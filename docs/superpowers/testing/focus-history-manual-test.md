# 月度专注记录功能手动测试清单

## 日历显示
- [ ] 进入“专注历史”页面后，日历保持 7 列 x 6 行布局，月份切换时高度不跳动。
- [ ] 每个有效日期格子上方显示日期数字，下方仅在有记录时显示当天专注总时长。
- [ ] 有记录日期的时长文字使用 11px 字号，颜色为 `#d4a574`。
- [ ] 无日期的空白格不响应点击，不出现透明黑色闪烁。
- [ ] 今天、悬停、选中状态的边框清晰可见，选中日期背景为 `#f0e6d2`。

## 时间轴显示
- [ ] 点击有专注记录的日期后，时间轴标题显示为“<月>月<日>日 专注记录”。
- [ ] 有记录时显示可滚动时间轴列表，每条记录都有左侧圆点和连接线。
- [ ] 每条记录显示任务标题、`HH:mm - HH:mm` 时间段、专注时长和“已完成”状态。
- [ ] 多条记录按开始时间从早到晚显示。
- [ ] 任务被删除或标题为空时，卡片标题显示“未知任务”。

## 空状态
- [ ] 点击无记录日期后，时间轴区域显示“这一天还没有专注记录”。
- [ ] 空状态文字居中显示，不挤压标题，不出现多余卡片。

## 交互
- [ ] 点击日期后，高亮日期立即切换，时间轴同步刷新为对应日期记录。
- [ ] 点击“上月”“下月”后，日历和时间轴数据同步刷新，选中日回到 1 号。
- [ ] 点击“本月”后，回到当前年月并选中今天。
- [ ] 记录较多时，时间轴可以纵向滚动，横向滚动条不出现。
- [ ] 鼠标悬停日期格子时背景变为 `#fffef9`，普通状态为 `#faf6ee`。

## 样式
- [ ] 日历、时间轴、卡片的圆角和边框与现有温暖纸质主题一致。
- [ ] 时间轴卡片间距稳定，滚动条颜色与页面配色一致。
- [ ] 窄窗口下任务标题和时长使用省略号，不与状态或时间段重叠。
- [ ] 页面中没有调试日志导致的控制台噪音。

## 边界
- [ ] 无任何专注记录的月份正常显示空日历和空状态。
- [ ] 只有 0 秒记录时，卡片时长显示为“0分钟”。
- [ ] `endTime` 缺失或为空时，时间段结束时间显示为 `--:--`，页面不崩溃。
- [ ] 闰年 2 月、30 天月份、31 天月份的前后空白格位置正确。
